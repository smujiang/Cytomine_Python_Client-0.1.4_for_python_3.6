This is a install pack of Cytomine_Python_Client-0.1.4 for python 3.6.
The original package is just runable on python 2.7, which can be cloned at:
https://github.com/cytomine/Cytomine-python-client
you may also find poster-0.8.1-for-Python-3.4 be useful, at:
https://github.com/mrd1no/poster-0.8.1-for-Python-3.4
=======
COMPILE
=======

python setup.py build

=======
INSTALL
=======

python setup.py install
